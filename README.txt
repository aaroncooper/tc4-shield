README.txt for /svn

Description of repository organization, as of April 4, 2011

/svn/applications:  applications software
/svn/hardware:  PCB designs and related information
/svn/wiki:  various wiki files
/svn/trunk:  deprecated original repository tree

All applications software, including utilities, demos, and libraries,
can now be found in /svn/applications.

All PCB designs and related information can now be found in
/svn/hardware.

aArtisan.pde
------------

Release 1.03

Supports Artisan 0.4.1 or Artisan 0.3.4.  Edit user.h to select Artisan version
to be supported.

Artisan command format:
  - "READ\n" for Artisan 0.3.4
  - "RF2000" or "RC2000" (no newline terminator) for Artisan 0.4.1

Current plans are to restore the newline terminator for future builds
of Artisan.  This version of aArtisan.pde will probably not work for
releases after Artisan 0.4.1.
